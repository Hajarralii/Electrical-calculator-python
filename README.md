# Electrical-calculator-python
Electrical Engineering Calculator built with Python to perform basic circuit calculations including current, voltage, resistance, and power.

## How to Run the Application

1. Make sure Python is installed
2. Download or clone this repository
3. Open the file in a Python IDE (PyCharm)
4. Run the file:

python gui_calculator.py

5. Use the buttons to select a calculation and enter values

## Features
- Graphical User Interface (GUI) using Tkinter
- Calculates Current, Voltage, Resistance, and Power
- Input validation and error handling
- Dynamic labels for user guidance
